Curriculum & Modules

Module 1 : Démarrage Professionnel avec Python
PDF

Maîtriser le workflow de démarrage d'un projet Python professionnel, mettre en place un environnement de développement stable et isolé (venv), appliquer la norme PEP 668 et les conventions de code PEP 8, utiliser les outils essentiels de qualité de code (Black) dans l'IDE.
Module 2 : Types de Données Fondamentaux et Opérateurs
PDF

Maîtriser les concepts de Variable et d'Affectation, Distinguer et manipuler les types de données de base de Python, Utiliser la syntaxe moderne des f-strings, Appliquer les opérateurs.
Module 3 : Les Structures de Contrôle et le Flux d'Exécution
PDF

Implémenter des logiques de décision, utiliser les boucles (for, while) pour gérer l'itération et la répétition des tâches, maîtriser les outils d'itération essentiels, Optimiser le flux d'exécution, choisir la structure de contrôle la plus appropriée.
